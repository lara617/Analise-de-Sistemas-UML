UML

UML (Unified Modeling Language) is a standardized graphical modeling language that is mainly used in the field of software engineering to visualize, specify, construct and document complex systems. UML was created to unify diverse modeling approaches and provide a common language understood by everyone involved in software development.

Crud refers to the four basic operations in relational database management systems or any system that involves data manipulation. CRUD operations are:
Create (C): Create, which involves creating new records or entities in the database. In programming terms, this usually means entering new data.
Read (R): Read, which involves querying and retrieving existing data from the database. This operation is often associated with reading information.
Update (U): Update, which involves modifying existing data in the database. This may include changing values in existing records.
Delete (D): Delete, which involves removing records or data from the database. In programming terms, this usually means removing information.
Open UML

Crud in our UML:
Create (C): Create the data for each book (title, author, year of release, genre, etc.), create the data for each person (name, telephone number, library identification number)
Read (R): Read the data for each book, the availability of each one (how many copies there are of each book)
Update (U): Update the information linking the user and the book (requests and returns), if there is a change in the user's personal data
Delete (D): Delete the user's association with the book.
