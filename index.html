UML

A UML (Unified Modeling Language) é uma linguagem de modelagem gráfica padronizada que é utilizada principalmente no campo da engenharia de software para visualizar, especificar, construir e documentar sistemas complexos. A UML foi criada para unificar diversas abordagens de modelagem e fornecer uma linguagem comum compreendida por todos os envolvidos no desenvolvimento de software.

Crud refere-se às quatro operações básicas em sistemas de gerenciamento de banco de dados relacionais ou em qualquer sistema que envolva manipulação de dados. As operações CRUD são:
Create (C): Criar, que envolve a criação de novos registros ou entidades no banco de dados. Em termos de programação, isso geralmente significa inserir novos dados.
Read (R): Ler, que envolve a consulta e a recuperação de dados existentes do banco de dados. Essa operação é muitas vezes associada à leitura de informações.
Update (U): Atualizar, que envolve a modificação dos dados existentes no banco de dados. Isso pode incluir a alteração de valores em registros existentes.
Delete (D): Excluir, que envolve a remoção de registros ou dados do banco de dados. Em termos de programação, isso geralmente significa remover informações.

Crud no nosso UML:
Create (C): Criar os dados de cada livro (título, autor, ano de lançamento, género, etc), criar os dados de cada pessoa (nome, telefone, número de identificação na biblioteca)
Read (R): Ler os dados de cada livro, as desponibilidade de cada um (quantos exemplares há de cada livro)
Update (U): Atualizar as informações que vincula o utilizador e o livro (requisições e devoluções), caso haja alteração nos dados pessoais do utilizador
Delete (D): Excluir a associação do utilizador ao livro.
